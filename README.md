# FitGirl Fast Downloader

> **Educational purposes only.** This project demonstrates targeted automation with Selenium.

---

## Usage & Scope

This project is a **targeted automation script**, intentionally limited to a specific workflow.

**Supported sources:**

- **Main site:** `fitgirl-repacks.site`
- **File host:** `fuckingfast.co`
- **Paste service:** `paste.fitgirl-repacks.site`

The script is designed **only** for this exact combination.  
Other hosts or mirrors are intentionally unsupported.

---

## How to Use

1. Open the application.  
2. When prompted, paste **only** a link in this format: https://paste.fitgirl-repacks.site/...
3. Press **Enter**.  
4. The program will automatically:
   - Extract all download links
   - Open each `fuckingfast.co` page
   - Trigger the download process without manual interaction

No additional input is required after submission.

---

## Important Notes

Pasting links from:

- the main FitGirl page  
- other paste services  
- other file hosts  

will **not work by design**.  

The automation relies on JavaScript-driven interactions specific to `fuckingfast.co`.
